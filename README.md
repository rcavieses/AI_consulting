# AI Consulting – GitHub Pages Website

A professional GitHub Pages website for an AI consulting business offering:

- **AI Training Courses & Capacitation** for managers, civil engineers, environmental engineers, lawyers, and SME owners — covering Claude, ChatGPT, and Gemini.
- **Specialized Software Development** for lawyers and managers (legal practice management, AI dashboards, chatbots, and more).
- **AI Strategy Consulting** for small and medium enterprises.

## Live Site

Enable GitHub Pages on the `main` branch (root folder) to publish the site at:  
`https://<username>.github.io/AI_consulting/`

## Project Structure

```
├── index.html        # Main landing page
├── css/
│   └── styles.css    # Responsive stylesheet
├── js/
│   └── main.js       # Interactivity (mobile nav, scroll, form)
└── README.md
```

## Features

- Responsive, mobile-friendly layout
- Sections: Hero, Services, Courses, Audiences, Software Dev, Why Us, Contact
- Accessible HTML with ARIA attributes
- No external dependencies — pure HTML, CSS, and vanilla JS
